Project created for the COMP 3710 class at Thompson Rivers University. Project contributors include Aidan Chan, Marvellous Ifezue, and Cody Soloveoff. 
Project goal is to generate chord progression that is intended to emulate the patterns in 1980s music using a genetic algorithm.
---
### Setup

1. Run `git clone https://github.com/Aidanjosiah02/chordmagedon.git`
2. Run the setup.bat or setup.sh file to create python venv and install dependencies.  
3. To activate your environment, run in the repo's root directory: .venv\Scripts\activate
4. run.py in the root will run what we have so far. The code basically shows how its used so far.
